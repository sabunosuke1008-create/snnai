# SNNAI Phase9 設計書：N-MNIST イベントデータ対応

## 概要

SNNAI v2 ロードマップの **Phase9: イベントデータ対応** を実装する。

## 目的

- イベントカメラ形式のデータ（N-MNIST）を SNN に入力できるようにする。
- フレーム画像から疑似イベント列を生成する変換器を実装する。
- イベント列をスパイク列に変換し、Phase8 の SNN 分類器で利用可能にする。

## 構成

- `snnai/benchmarks/nmnist_loader.py`: N-MNIST 形式のイベントローダー・疑似イベント生成
- `snnai/benchmarks/event_utils.py`: イベント列 ↔ スパイク列変換
- `tests/test_phase9.py`: イベントデータの読み込み・変換テスト

## 疑似イベント生成

MNIST の連続フレームを想定し、画素値の変化量に応じて ON/OFF イベントを生成する。
- 画素値が上昇 → ON イベント
- 画素値が下降 → OFF イベント
- 変化が小さい → イベントなし

## 評価指標

- 生成されたイベント数が適切な範囲にあること
- イベント列からスパイク列への変換が正しい形状であること
