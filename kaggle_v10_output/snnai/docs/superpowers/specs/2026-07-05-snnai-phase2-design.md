# SNNAI Phase2 設計書：線虫モジュール

## 概要

SNNAI ロードマップの **Phase2：線虫モジュール（反射・最低限制御）** を実装する。本モジュールは **Reflex / Event Detection / Filtering** を目的とする。

## 背景

- C. elegans は約 302 ニューロンからなる。本モジュールでは 20〜50 ニューロンの「反射弓サブセット」を模擬する。
- Phase1 で整備した LIF/Izhikevich/STDP/LSM を基盤とする。

## 目標

1. `snnai/modules/c_elegans/` に反射モジュールを実装する。
2. 単純な走性行動（刺激に対する回避/接近）を模倣する。
3. 反射応答の遅延が既定閾値以下、かつ誤反応率が一定以下であることを確認する。

## 設計

### モジュール構造

```
snnai/modules/
└── c_elegans/
    ├── __init__.py
    ├── reflex_module.py
    └── tests/test_reflex_latency.py
```

### 反射モジュール

- **入力**: 感覚ニューロン（例：左/右の刺激強度）
- **中間**: 興奮性/抑制性中間ニューロン
- **出力**: 運動ニューロン（左/右への行動）
- **学習**: STDP による反射強度の調整

### タスク

OpenAI Gym 的な簡易環境：
- 1D 空間上でエージェントが左右に移動
- 左側に刺激が来たら右へ回避、右側に刺激が来たら左へ回避
- 反射応答遅延を測定

### 評価指標

- 反射応答遅延 < 5 ms（シミュレーション時間）
- 誤反応率 < 10%

## 実装方針

- snnTorch ベース。
- `snnai/core/neurons/lif.py` と `snnai/core/synapses/stdp.py` を利用。
- 環境は簡易的な Python クラスで実装。

## 次フェーズ条件

- 反射モジュールが他モジュールと独立に動作すること。
- 割り込み動作（他処理を中断して反射を優先）が実装できること。
